# SteamHourboost
### A Steam game hour idling software!


**Features:**
* Fully automatic
* Multi-account support
* Per-account game selection

**Main Libs:**
* [node-steam-user](https://github.com/DoctorMcKay/node-steam-user)
* [Steam-ToTp](https://github.com/DoctorMcKay/node-steam-totp)
